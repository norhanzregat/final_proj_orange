Crankshaft


“In this design, I created a 3D model using Autodesk Fusion 360. The model consists of a green cylinder connected to an orange arm. At the end of the arm is a component that looks like a wheel or pulley. When the wheel is turned, the cylinder inside the model moves. I used this design to simulate a certain mechanical movement, where I can see how the different parts interact with each other. This type of modeling helps me make sure that the design works correctly before moving on to the actual manufacturing stage.

I created a 3D model of a cup using Fusion 360's Revolve tool. The design began with a basic sketch of the cup's profile on a 2D plane. Using the Revolve tool, I rotated this profile around a central axis to form the full 3D shape. This approach allowed me to precisely control the shape of the cup, ensuring it has smooth curves and an even surface. Fusion 360’s features also let me fine-tune dimensions and view the model from all angles, ensuring the final design met my specifications.

I designed a 3D model of a diamond using the Polygon tool in Fusion 360. I started by drawing a 12-sided polygon. I then used transform tools like Extrude to transform this polygon into a 3D diamond-like shape. This method helped me to form the facets accurately and symmetrically. I was also able to adjust the dimensions and view the design from all angles to ensure the final shape was up to the specifications.

I designed a 3D hairbrush using Fusion 360. I started by drawing the oval shape of the brush head using the Sketch tool, then converted it into a 3D model using the Extrude tool. Next, I drew the long handle of the brush attached to the head, and added it using the same tool. I used tools like Offset to create additional layers for the brush head to accurately define the dimensions. Finally, I used the Construct tool to adjust the position of the handle in relation to the brush head and ensure the design was balanced. These steps allowed me to control the dimensions of the brush and review it from all angles to ensure the final design was consistent with the requirements.

I designed a 3D circular frame using Fusion 360. I started by drawing the basic outline of the frame as a circle, then used the Extrude tool to convert it into a 3D shape with an inner cavity. I then added two raised edges on the edges using the Offset tool to duplicate the layer and adjust the thickness of the edges. Finally, I adjusted the settings to make the inner part transparent, to highlight the cavity and highlight the design details. This frame can be used in industrial or engineering applications, with the dimensions easily adjustable to suit different needs.
